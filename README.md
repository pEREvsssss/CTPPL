# Clearly Team Project Protection License (CTPPL)

Version 1.0

## Overview

The **Clearly Team Project Protection License (CTPPL)** is a custom license designed to protect projects developed by Clearly Team (maintained by KilixKilik, pEREvsssss). This license allows for open-source distribution while ensuring proper attribution and protecting the original author's interests.

## Key Features

- **Permissive for Use**: Allows commercial and non-commercial use.
- **Attribution Required**: Must credit "Clearly Team" and the original author "KilixKilik".
- **Modification Allowed**: Permits modification of the source code.
- **Distribution Permitted**: Allows redistribution under the same license terms.
- **Clear Terms**: Defines conditions and restrictions for usage.

## License Text

The full legal text of the CTPPL can be found in the `LICENSE` file included with this project.

## How to Apply This License

To use the CTPPL for your project:

1.  Include a copy of the `LICENSE` file in your project repository.
2.  Add a brief notice in your project's `README.md` (like this one).
3.  Optionally, add a comment header to the top of your source code files:
    ```text
    /*
    * Licensed under the Clearly Team Project Protection License (CTPPL).
    * See LICENSE file for details.
    * Copyright (c) Clearly Team (KilixKilik).
    */
    ```
